Windows批处理运行gulp压缩前端文件（自用）

最近转岗了,干起了实施，制造业嘛，还是要融入业务的，那就发挥精神哪里需要哪里填呗(主要是没人…)。现阶段是事情杂，时间零碎，所以就搞搞gulp，顺便弄弄windows批处理啥的。

1.进入到gulp盘后，执行gulp clean后，命令窗口就关闭了，下面的其他指令并不执行。eg：

@echo off
echo 进入E盘
e:
echo 进入E盘gulp-pack文件夹
cd gulp-pack
echo 执行gulp clean
gulp clean
echo 完成
pause

当执行完gulp clean后窗口自动关闭，其后的pause并不执行。这是因为执行完 gulp clean命令后这个命令接收器的实例就结束了，窗口就关闭了。因此需要启动一个新实例来完成该命令。因此需要改写成如下：

@echo off
echo 进入E盘
e:
echo 进入E盘gulp-pack文件夹
cd gulp-pack
echo 执行gulp clean
cmd /c gulp clean
echo 完成
pause

此时相当于在该路径下重新开了个cmd窗口，然后执行 gulp clean，/c 执行字符串指定的命令然后终止，cmd后的参数有 /c、/k、/s、/q 等，输入 cmd/? 查看帮助。注意，cmd 后的 参数必须要有，不然只是执行了 cmd 后面的命令是不执行的。

2.cmd 更换盘符

必须是 e: 然后执行 cd xxx 才能进入相关目录，执行 cd e:/xxx 不能进入，需要使用 /D 参数,即 cd /d e:\xxx。使用 /D 开关，除了改变驱动器的当前目录之外，还可改变当前驱动器。用 cd/? 查看帮助。

3.copy 复制文件

copy /y e:a.text e:\gulp-pack\ 将e盘下的a.text 复制到e盘 gulp-pack 目录下。/y 取消提示以确认要覆盖现有目标文件
copy /y e:\a.txt+e:\b.txt e:\gulp-pack\ 将e盘下的a.text和b.text 合并为一个文件并 复制到e盘 gulp-pack 目录下。
copy 不能复制多个文件，需要分开多条命令复制。且只能复制文件，对于文件夹的复制需要用xcopy来。更多参数 用 copy/? 查看帮助

4.xcopy 复制文件夹

xcopy e:\temp e:\gulp-pack\src\ /e /y 复制e盘下temp文件 /e 复制目录和子目录，包括空目录。 /y 取消提示以确认要覆盖现有目标文件
注意 源文件的路径不能以 \ 结尾 即不能使用 e:\temp\
如果需要排除文件夹下的一些文件，则需要使用 /exclude:nocopy.txt 在nocopy.txt中写入不复制的文件，如果nocopy.txt不在当前目录，则加上路径
eg:/exclude:e:\nocopy.txt。最好直接写绝对路径。更多参数 用 xcopy/? 查看帮助

5.rmdir 删除目录

rmdir /s/q e:\temp 删除e盘下temp目录 /s 删除所有子目录。 /q 不问我 Yes or No 的动作。

6.批处理（自用 ）

@echo off

echo 删除E盘下gulp-pack文件夹src

rmdir /s/q e:\gulp-pack\src

创建src文件夹

makdir e:\gulp-pack\src

echo 将前端内容复制到src目录下

xcopy E:\work\platform\src\main\webapp e:\gulp-pack\src\  /e /y

echo 删除不相关文件

del /q e:\gulp-pack\src\index1.html e:\gulp-pack\src\productSearch.html e:\gulp-pack\src\jSignature.html

echo 进入E盘gulp-pack文件夹

cd /d e:\gulp-pack

echo 执行gulp build

cmd /c gulp build

echo 完成

pause
